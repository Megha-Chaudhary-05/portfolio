Movie Recommendation
System

Built a content-based movie recommendation engine using TF-IDF vectorisation on genre, cast, and plot keywords from the TMDB dataset. Cosine similarity is used to identify the top-10 most similar movies to any given title. The project also explores collaborative filtering using matrix factorisation to surface user-personalised suggestions beyond genre boundaries. The final model was evaluated with precision and recall metrics, achieving 78 % precision@10 on the test split.

Dataset TMDB 5000

Python Scikit-learn Pandas NumPy Classification

Customer Churn
Prediction

Developed a binary classification pipeline to predict whether a telecom customer will churn within the next 30 days. The project covers the full ML lifecycle — data cleaning, feature engineering (tenure buckets, contract flags, average monthly spend), model training with Logistic Regression and Random Forest, hyperparameter tuning via GridSearchCV, and evaluation using ROC-AUC and F1 score. The Random Forest model achieved 91 % accuracy and 0.94 AUC, surfacing contract type and tenure as the two strongest churn predictors.
